Inline text elements

The <a> element is for anchor links.

The <em> element is for emphasised text, and <strong> is for important text

The <i> element is for text in an alternative voice or a technical term

The <b> element is for text like product names in a review, or other spans of text whose typical typographic presentation is bold

The <q> element is for inline quotes.

The <code> element is for computer code.

The <ins> element is for marking inserts to a document, and the <del> element is for deletions.

The <mark> element is used to highlight text

The <span> element is used for text with no specific meaning. This has to be used when no other text-semantic element conveys an adequate meaning

The <sup> element is used for superscript and the <sub> element is for subscript

The <abbr> element is used for abbreviations or acronyms text
